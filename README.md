# Demo
[portofolio-ecommerce.vercel.app](https://portofolio-ecommerce.vercel.app/)

# Installation
- Clone project
- Run `yarn`
- Run `yarn dev`
- Open `localhost:3000` in browser

# Development
- Run `yarn dev`
- Open `localhost:3000` in browser

# Production Build
- Run `yarn build`
- Run `yarn start`
- Set environment type on .env
- Open `localhost:3000` in browser
